# Best Buy 2
The second part of a Shopping Demo Project for MSIT Masterschool
It is a Shop where you can buy items in the CLI.

## Usage
- run `python3 main.py`

## Products
- Normal products, Non stockes products or limited products
- Allow price comparison with `<` and `>`
- Magic Method `__str__` instead of `show()`
## Store
- Stores products
- Possible to combine 2 stores into one by adding
- Check if products in store with in method
## Promotions
- 3 methods implemented, more possible
- `SecondHalfPrice` halfes price for every second bought item of product
- `ThirdOneFree` every third bought item of product is free
- `PercentDiscount` discount for every item of product by selected percent value
